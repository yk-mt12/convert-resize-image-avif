# convert-resize-image-avif
- 画像ファイルサイズの縮小します。
- 画像をアスペクト比を固定しつつ任意の拡張子に変換します。

## 環境
```
venv
Python 3.12.3

```
## 実行方法
```
# venvを使用する場合
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python convert.py

# venvを使用しない場合
pip install -r requirements.txt
python convert.py
```
